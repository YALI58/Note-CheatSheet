每个HTML元素在页面上都是一个"盒子"。盒模型就是定义这个盒子如何**占据空间**的规则。
```css
/* 盒模型公式 */
总宽度 = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
总高度 = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom
```

## 盒模型的两种模式：`content-box` vs `border-box`
```css
		.box {
            width: 300px;
            padding: 40px;
            border: 20px solid #2196F3;
            background-color: #FFEB3B;
            font-weight: bold;
            text-align: center;
        }
		.content-box-example {
            /* 默认值：标准盒模型 */
            box-sizing: content-box;
        }
        
        .border-box-example {
            /* 现代更常用的方式 */
            box-sizing: border-box;
        }
```
border-box 规定的 box 的总宽度就是width：300px 不会因为padding和border而改变

content-box 则是会加上

以后写CSS时最好加上
```css
*{
	box-sizing : border-box
}

这是全局设置
```
