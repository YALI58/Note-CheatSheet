### **Windows Terminal（Windows）**

Windows Terminal 从 v1.4 起支持原生分屏。

|功能|快捷键（默认）|
|---|---|
|**垂直分屏**|`Alt + Shift + +`（加号）|
|**水平分屏**|`Alt + Shift + -`（减号）|
|**切换分屏**|`Alt + ← / → / ↑ / ↓`|
|**关闭当前分屏**|`Ctrl + Shift + W`|
|**调整分屏大小**|`Alt + Shift + 方向键`|

### **Windows**

#### **Windows Terminal（推荐）**

| 快捷键                                 | 功能                 |
| ----------------------------------- | ------------------ |
| `Ctrl + Shift + T`                  | **新建标签页**（默认配置）    |
| `Ctrl + Shift + W`                  | 关闭当前标签页            |
| `Ctrl + Shift + T`（关闭后）             | **重新打开最近关闭的标签页** ✅ |
| `Ctrl + Alt + ← / →`                | 切换到上一个 / 下一个标签页    |
| `Ctrl + 1~9`                        | 切换到第 1~9 个标签页      |
| `Ctrl + Tab` / `Ctrl + Shift + Tab` | 循环切换标签页（需在设置中启用）   |
