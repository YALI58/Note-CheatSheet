# 可能暴露信息的常见途径（按风险从高到低）

- **[上传/联网]**  
    程序如果把文件内容、文件名、路径、机器标识等通过 HTTP/WebSocket 发到外部服务器，就可能泄露。常见位置：`requests`/`httpx`/`urllib`、`socket`、`websocket`、调用第三方 API（如云识别/LLM）。
    
- **[日志与错误上报]**  
    报错时把**本机用户名、绝对路径、文件名**写进日志，或把异常堆栈上报到 Sentry/类似服务。
    
- **[硬编码密钥/Token]**  
    代码里包含 API Key、数据库密码、第三方服务 Token。即使你没“主动上传”，只要把仓库发出去/开源了也会暴露。
    
- **[打包产物携带隐私]**  
    打包（PyInstaller/Electron 等）时把 `.env`、配置文件、测试数据、缓存文件一起打进发布包。
    
- **[模型/数据集隐私]**  
    如果训练数据里含个人文件名/路径/内容，或者模型文件里包含可还原的敏感数据（较少见但可能）。
